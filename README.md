# React + Vite

Progetto di tirocinio per ingegneria informatica e automazione. Riguarda la realizzazione di uno spazio libero per la creazione di un Workflow formato da blocchi per organizzare le operazioni da applicare ad un data set in ingresso, che può essere di vario tipo: dati probenienti da sensori, macchine, fonti varie. Il progetto è realizzato tramite React e in particolare con l'utilizzo di una libreria chiamata React Flow che mette a disposizione una serie di funzionalità predisposte per la realizzazione, gestione e manutenzione di uno o più Workflow di lavoro. Di conseguenza il linguaggio utilizzato è HTML con Javascript e CSS.
